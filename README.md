#  Aplicação de Galeria de Fotos -Projeto

**Veja o projeto online aqui:** https://mariaems99.github.io/Projeto---Aplica-o-de-Galeria-de-Fotos/

Este projeto é uma aplicação web simples e funcional que permite visualizar uma galeria de imagens com busca em tempo real. 

##  Funcionalidades

- **Visualização em Grid**: Exibição de fotos em um layout organizado e moderno.
- **Busca em Tempo Real**: Filtro instantâneo conforme o usuário digita no campo de pesquisa (baseado no nome da foto).
- **Interatividade**: Efeito visual de aumento de escala (*zoom*) e sombra ao passar o mouse sobre as imagens.
- **Responsividade**: O layout se adapta automaticamente a diferentes tamanhos de tela (Mobile, Tablet e Desktop).
- **Feedback visual**: Mensagem informativa caso nenhuma foto seja encontrada na busca.

##  Tecnologias Utilizadas

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização, layout em Grid e efeitos de transição.
* **JavaScript**: Manipulação dinâmica do DOM e lógica de filtragem.

## Como rodar o projeto localmente

Como o projeto utiliza apenas tecnologias nativas, não é necessário instalar dependências 

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/mariaems99/Projeto---Aplica-o-de-Galeria-de-Fotos.git
    ```
2.  **Acesse a pasta do projeto:**
    ```bash
    cd (https://github.com/mariaems99/Projeto---Aplica-o-de-Galeria-de-Fotos.git)
    ```
3.  **Abra o arquivo principal:**
    Basta dar um duplo clique no arquivo `index.html` ou abri-lo através do seu navegador de preferência.

## Detalhes Técnicos (Requisitos)

- Cabeçalho com título centralizado.
- Barra de busca acima do grid.
- Grid de fotos com pelo menos 10 itens.
- Filtro funcional por nome da foto.
- Rodapé com informações de copyright de 2026.
- Efeito Hover nas fotos.
